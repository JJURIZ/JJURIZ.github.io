Kaleidoscope

MongoDB, Express, React, Node.js, SASS
Role: Front End Developer
Team: Alan Avery, Jackson Reeves, and Thomas Gilbert
Live Site GitHub

Kaleidoscope is a macro level budgeting MERN application that provides users with a quick way to see how small budget changes add up.

Our objective was to create an visually stimulating view of a typically dull subject: budgeting. Giving users the freedom to create custom budget items within fixed categories provides flexibility for all users.

My primary contribute was to assist with the frontend work. I assisted with implementing Recharts and React Alerts. When possible we practiced pair programming and managed user stories in Trello.
